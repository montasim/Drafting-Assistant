# Request public-source permissions on demand

Public Source Research will be disabled on a fresh installation even though all five approved source choices are preselected as inactive defaults; returning users recover their saved research and source-selection state, and configured-state visual fixtures may show the prototype's enabled state. Turning research on requests the currently selected source origins together in one Chrome permission prompt; declining returns the master switch to off without changing the selections. While research is on, selecting another source requests only its origin, deselecting one removes its host permission immediately, and external revocation disables only the affected source while other permitted sources remain usable. Turning the master switch off disables every source adapter and prevents all public-source requests but retains previously granted host permissions and source selections so re-enabling does not cause unnecessary prompts; the user may remove a grant by deselecting that source or through Chrome. Thoughtline tracks each grant independently and continues operating with partial permission while enabled. When research is disabled, **New search** opens the approved prototype's single **From your experience** fallback immediately without a source request or AI call; an AI job starts only after the user supplies one real lesson and chooses **Create evergreen post**. When research is enabled, the same fallback appears only after no enabled source produces a qualifying Idea. It can draft an unsourced post from the supplied lesson and saved writing profile, requires no public-source permission, and Thoughtline will not introduce a separate automatically generated Evergreen Ideas list. The resulting editor preserves the approved post-card layout and actions but uses truthful experience-specific content: a **From your experience** badge, a **Your lesson** card containing the submitted lesson verbatim without language tabs, and **Source: Your experience** inside **Source and writing direction**; it never labels personal input as a LinkedIn or external-source summary.
