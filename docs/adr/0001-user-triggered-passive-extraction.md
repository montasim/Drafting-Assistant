# Use user-triggered passive post extraction

The Drafting Assistant will identify one post through Chrome's context menu and extract only its already-rendered context after an explicit user request. This provides a better experience than manual pasting even though LinkedIn broadly prohibits extensions that scrape or copy its services and may restrict accounts; the user accepts that non-zero risk. The product will not load hidden content, modify LinkedIn's DOM, automate LinkedIn actions, submit drafts, or attempt to evade LinkedIn's defenses.
