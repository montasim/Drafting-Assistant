# Make discovery a separate opt-in capability

New-user onboarding will ask whether the user wants discovery and will show Groq consent, credential, source, topic, and optional voice setup only after an affirmative choice. Existing users will not be returned to onboarding or prompted automatically after an upgrade; they may enable the capability later from Settings, and declining or revoking Discovery Consent affects neither the existing Gemini response workflow nor LinkedIn permissions.
