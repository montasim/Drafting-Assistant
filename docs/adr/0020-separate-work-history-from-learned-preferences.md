# Separate work history from learned preferences

Clearing Work History will remove Reply, Idea, and Rewrite records together with their raw Feedback Evidence, but it will not erase the compact Learned Writing Preferences already derived from that evidence. Resetting learned preferences removes the accepted preference summary and deterministic ordering/feature scores but leaves raw Feedback Evidence available for a later explicit re-analysis; only clearing the associated Work History removes that evidence. Manually entered profile, tone, and style guidance are unaffected by either operation.
