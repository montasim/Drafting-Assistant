# Use WXT, React, and TypeScript

Thoughtline will use WXT with React 19 and TypeScript for its Manifest V3 background worker, LinkedIn content script, onboarding page, and side panel. Tailwind CSS v4 will provide the tokenized styling layer and shadcn/ui with Radix primitives will provide accessible, locally owned component foundations; this combination supports the required modular boundaries while allowing exact reproduction of the approved prototype.
