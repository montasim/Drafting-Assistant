# Require a minimum writing profile during onboarding

Thoughtline reaches Setup Readiness only after the user has affirmatively granted AI Processing Consent, allowed LinkedIn page access, validated both required Provider Credentials, and supplied non-empty, Zod-valid values for **Your role**, **Topics you know well**, and **People you want to reach**. The selected default tone may remain unchanged, and Profile Import remains optional because it only proposes editable values for those fields. Each setup step preserves valid completed work when the user navigates backward, but the final **Start writing** action remains unavailable until every required boundary is satisfied; the readiness screen and Settings checker report the incomplete category rather than silently filling profile answers. The **Writing profile** section provides a right-aligned secondary **Clear profile** action with a default-safe confirmation; confirmation removes those three fields and any retained profile-derived data without deleting Work History, Provider Credentials, the separately managed Style Guide, Learned Writing Preferences, or unrelated settings. Clearing the profile immediately makes Setup Readiness incomplete and blocks new Foreground AI Jobs until all three required fields are supplied again, while local History, editing, copying, and Settings remain available.
