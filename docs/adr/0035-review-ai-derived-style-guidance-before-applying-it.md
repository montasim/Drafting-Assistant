# Review AI-derived style guidance before applying it

**Create style guide** starts a provider-neutral Foreground AI Job using the writing samples the user deliberately submits. A successful result opens an editable preview and requires explicit **Apply style guide** or **Cancel** actions; applying atomically replaces the current Style Guide, while cancellation, validation failure, provider failure, or panel closure leaves the saved guide unchanged. Submitted samples are transient request inputs and are discarded after generation unless the user explicitly chooses to retain them as part of the writing profile. The resulting guide remains directly editable in Settings and follows the approved instruction precedence rather than becoming hidden provider training or a Learned Writing Preference.
