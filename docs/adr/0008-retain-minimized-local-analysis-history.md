# Retain a minimized local analysis history

The extension will keep the 20 most recent History Entries in extension-local storage and delete the oldest when adding another. Each entry retains only creation time, Response Target type, a short post excerpt, Analysis Summary, Draft Set, Draft Language, and model identifier; complete posts, visible discussions, Engagement Profile snapshots, member identifiers, and raw prompts are not retained, trading exact reproducibility for lower privacy exposure and predictable storage use.
