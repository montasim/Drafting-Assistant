# Minimize and de-identify post context before transmission

The extension will send Gemini only the selected post and discussion text needed for response drafting, aggregate reaction information, and temporary Participant Labels. It will remove names, profile URLs, member identifiers, reactor identities, headlines, images, video, and unrelated profile information before transmission, accepting less context in exchange for lower privacy exposure, fewer prompt tokens, and a clearer public privacy promise.
