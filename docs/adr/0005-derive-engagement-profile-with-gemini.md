# Derive the engagement profile from a raw PDF with Gemini

After explicit disclosure, the extension will send the user's own complete Profile PDF directly to Gemini to derive an editable Engagement Profile, rather than parsing the document locally. The raw PDF and provider response will be discarded after review, only the user-approved Engagement Profile will persist, and contact details will be excluded from that profile; this accepts greater one-time privacy exposure in exchange for simpler and more adaptable document understanding.
